### Unity3D中的ML-Agents插件使用说明

Unity Machine Learning Agents (ML-Agents) 是一款开源的 Unity 机器学习插件， 使得我们得以在游戏环境和模拟环境中训练智能 agents。它可以使用 reinforcement learning（强化学习）、imitation learning（模仿学习）、neuroevolution（神经进化）等机器学习方法，通过简单易用的 Python API进行控制，对 Agent 进行训练。

这部分内容主要收录一下本人使用ML-Agents中经常使用到的一些功能和注意点，具体的插件使用说明见原仓库链接
